# Changelog

All notable changes to the Yozakura theme extension will be documented here.

## [1.0.0] - 2026-05-10

### Added
- **Yozakura Yoru** — deep navy dark theme with blush, lavender, and gold accents
- **Yozakura Hiru** — soft lavender light theme with violet, rose, and amber accents
- Full UI chrome coverage: editor, sidebar, activity bar, tabs, status bar, panels, terminal
- Terminal ANSI color mapping for both variants
- Syntax highlighting for JavaScript, TypeScript, HTML, JSX, CSS, JSON, and Markdown
- Git decoration colors for both variants
- Debugger and symbol icon color mappings
- Bracket pair colorization support
